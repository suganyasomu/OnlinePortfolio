This is a single-page application that I have completed for the bootcamp from university of Oregon. This project was finished using the Javascript framework javascript.js,CSS and HTML.

The page is loaded with the daily event schedule for office hours 9am to 5pm . User should be able to create an event and save it.The past hours should be displayed in grey , current hours should be in green and the future hours should be in green.
