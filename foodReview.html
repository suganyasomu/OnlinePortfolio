Food Review Project

  • Javascript
  • Ajax
  • Jquery
  • Yelp API
  • Node.js
  • Express
  • Sequelize
  • heroku

This is a single-page application that I have completed for the bootcamp from university of Oregon. This project was finished connecting the the Frontend client and backend server using javascript, Node.js,express and sequelize and the external "Yelp api".

The page is loaded with the list of top 10 restaurants in your area from the yelp API. when the user clicks on the restaurant link,it navigates the user to the actual web site of the restaurant. Two buttons are displayed next to the each restaurant link. When the user clicks on the "Review a Dish" button ,it navigates the user to the new page where the user can add a review to existing dish or a new dish and when the user clicks on the "View reviews" button, it navigates the user to view the existing reviews for the selected restaurant.

The page is responsive and it is intended for mobile viewing on small devices.
