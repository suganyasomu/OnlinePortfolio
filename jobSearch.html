This is a single-page application that I have completed for the bootcamp from university of Oregon. This project was finished using the Javascript framework javascript.js and a third-party job search API .

The page is loaded with the search criteria for job title and location . When the user search with specific criteria , the page is loded with the top 5 results on the left side and when the user clicks on any result, More details should be displayed on the right side . User should be able to apply for the position by clicking on the "Apply" link .
