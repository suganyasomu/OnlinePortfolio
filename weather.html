Weather API Project

  • Javascript
  • Ajax
  • Jquery
  • Weather API

This is a single-page application that I have completed for the bootcamp from university of Oregon. This project was finished using the Javascript framework javascript.js and a third-party Weather API .

The page is loaded with the default place weather for the current day and forecast data for the next 5 days . User can search with any city or state in the united states on the left side which displays the current weather of that place and forecast for next 5 days.

The page is responsive but it is not intended for mobile viewing on small devices. If you are viewing on a mobile phone, landscape is recommended as the width of the search list is fixed.
