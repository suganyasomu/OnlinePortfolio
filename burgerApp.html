This is a single-page application that I have completed for the bootcamp from university of Oregon. This project was finished using the Javascript framework javascript.js and node.js CLI, express and mysql in MVC paradigm

The page is loaded with the input criteria with the submit button.The user can enter the new burger name and click on the submit. The newly added burger will be added under the "not devoured" list with the "devour it" button and when the user clicks on the "devour it" button, the value should be updated under the "Are devoured" list on the right side of the page.
